# 杨辉三角-简单

难度：简单

题目描述：
给定一个非负整数  *numRows，*生成杨辉三角的前  *numRows *行。

在杨辉三角中，每个数是它左上方和右上方的数的和。

示例：

输入: 5;
输出: [[1], [1, 1], [1, 2, 1], [1, 3, 3, 1], [1, 4, 6, 4, 1]];
1
2

解题思路：
除第一行外，第 n 行第 j 个都等于(n-1)[j-1]+(n-1)[j]

var generate = function (n) {
  let i = 0;
  let data = [];
  while (i < n) {
    let arr = [];
    if (i === 0) {
      arr[i] = 1;
    } else {
      for (let j = 0; j < i + 1; j++) {
        let pre = data[i - 1][j - 1] || 0;
        let cur = data[i - 1][j] || 0;
        arr[j] = pre + cur;
      }
    }
    data.push(arr);
    i++;
  }
  return data;
};
